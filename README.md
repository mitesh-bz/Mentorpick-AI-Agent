# Mentorpick AI Agent

A highly scalable, real-time autonomous AI mentor built for the Mentorpick NodeBB forum. This agent actively monitors the "Ask Doubt" category, fetches student code submissions, and utilizes Google Gemini to provide empathetic, personalized, and context-aware hints without giving away direct answers.

## 🛠️ Prerequisites
- [Node.js](https://nodejs.org/) (v16.x or higher recommended)
- `npm` (Node Package Manager)

---

## 🚀 Run Agent (Development)

To run the application locally for testing:

Create .env file with required fields mentioned in .env.example

```bash
npm install
npm run dev
